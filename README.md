# PLANEXA 🚀

Planexa is a modern task management web app designed to help users organize their daily tasks efficiently.

## ✨ Features
- Add tasks with time ⏰
- Mark tasks as completed ✅
- Delete tasks 🗑️
- Feedback system 💬
- Clean and modern UI 🎨

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
